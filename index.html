Greek Software Ecosystem

The open-source pulse on IT and software jobs across Greece

Greek Software Ecosystem is a community-maintained collection for people exploring software and technology careers in Greece. Among other things, this repository holds a sortable employer directory (with weekly Workable open-role snapshots where applicable), curated job-search links and tips, pointers to Greek tech…

Disclaimer

This project is for community and educational purposes only. Data is aggregated from public sources (company websites, LinkedIn, and Workable public API). Job counts are cached snapshots and are not a real-time official database.

Non-affiliation: this project is independent and not affiliated with, sponsored by, or endorsed by any listed company. Remote, hybrid, and on-site labels are crowdsourced and may be wrong—always verify openings, work arrangements, and policies on each employer's official website using the listed URLs.
